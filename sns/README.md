# BUZZ BASE SNS運用

BUZZ BASEの認知拡大・ユーザー獲得を目的としたSNS運用に関するドキュメントとデータを保管するディレクトリ。

## 運用方針

**個人ブランディング型**：BUZZ BASEの公式アカウントではなく、開発者「Ippei」個人のアカウントとして運用する。

- メインSNS: **X (旧Twitter)** — `@ippei_buzzbase`
- ターゲット: 中高生・大学生・社会人野球選手 / 保護者 / 指導者 / 個人開発者コミュニティ
- 訴求軸: 「16年野球を続けた個人開発者が、野球選手のために作った成績記録アプリ」
- ピボットコストを下げるため、user ID にはアプリ名を含むが個人として運用する

## ディレクトリ構成

```
sns/
├── README.md          # このファイル
└── x/                  # X (旧Twitter) 専用
    ├── profile.md      # プロフィール設定の最新版
    ├── pinned-post.md  # 固定ポスト
    ├── strategy.md     # 運用戦略・KPI・コンテンツ設計
    └── follow-targets.md # フォロー対象100アカウント計画
```

将来、Instagram・TikTok等を運用する際は `instagram/` `tiktok/` ディレクトリを追加する。
