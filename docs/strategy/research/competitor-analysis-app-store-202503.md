## 競合が共通して使っている訴求ポイント

1. **無料** または **基本無料** - ほぼ全競合が無料を前面に打ち出す
2. **簡単入力・直感的操作** - 入力の手軽さは共通訴求
3. **成績の自動計算** - 打率・防御率などを自動算出
4. **データ可視化・グラフ** - 視覚的にわかりやすい表示
5. **セイバーメトリクス対応** - OPS・WHIPなどの高度指標
6. **チームへの共有・速報** - 観客・保護者との情報共有

## BUZZ BASE の差別化ポイント（競合にない独自優位性）

### 1. 「ランキング形式での成績比較・共有」- 競合ゼロ

個人成績をランキング形式で友人・チームメイトと比較できる機能を持つアプリは調査した競合に存在しない。PLAY by TeamHub にチーム内成績表示はあるが、複数チームをまたいだグループでのランキング比較はできない。

### 2. 「チーム横断グループ機能」- 競合ゼロ

所属チームが異なる選手同士でもグループを作れる設計。他アプリは同一チーム内での共有が基本。同じ中学生・高校生でも他チームのライバルと成績を競い合える。

### 3. 「29種類の成績指標を自動計算」- 最高水準

打撃7種（打率・出塁率・長打率・OPS・ISO・ISOD・BB/K）+ 投手22種（防御率・K/9・WHIP・FIPなど）= 合計29種。打撃のみに特化した競合（ヒットメーカー）より投手成績の網羅性が高い。

### 4. 「モチベーション訴求」- 競合は管理・記録に留まる

競合は「記録する・管理する」という機能訴求が中心。BUZZ BASEは「友達と競い合う」という競争・モチベーション向上の体験価値を提供している。

## 市場規模参考データ

### 日本の中高生野球人口（2024年時点）

| カテゴリ | 人数 |
|---------|------|
| 高校野球部員（硬式） | 127,031人 |
| 高校野球部員（軟式） | 7,716人 |
| 中学軟式野球（中体連） | 133,725人 |
| 中学硬式野球（リトルシニア等） | 推定3〜5万人（複数団体） |
| **中高生野球人口合計（推計）** | **約30〜35万人** |

10代全体の野球実施人口は約174万人（笹川スポーツ財団調査）。

### 市場トレンド

- 野球人口は減少傾向（高校硬式部員は10年連続減少）
- ただし中学硬式野球は少子化比で相対的に安定
- デジタル・データ活用志向の高まり（セイバーメトリクスへの関心増加）
- スマートフォン利用率の上昇（中高生のスマホ保有率は90%超）

## App Store 掲載テキスト設計への示唆

### 使うべきキーワード（競合が使用 + 検索需要あり）

`野球` `成績` `記録` `打率` `防御率` `OPS` `WHIP` `個人成績` `ランキング` `グループ` `自動計算` `無料`

### BUZZ BASE 独自のキーワード候補

`ランキング共有` `成績比較` `仲間と競う` `グループランキング` `チーム横断`

### プロモーションテキスト方向性

競合は「記録・管理・分析」の機能訴求が中心。BUZZ BASE は「仲間と競い合う・モチベーションが上がる」という体験価値で差別化できる。

**候補フレーズ例:**
- 「打率・防御率をランキングで競い合う。野球がもっと燃える。」
- 「29種の成績を自動計算。友達とランキングで比較できる唯一の野球アプリ。」
- 「記録するだけで終わらない。ランキングで仲間と成績を競い合おう。」
