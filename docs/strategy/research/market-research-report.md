**作成日**: 2026年3月14日
**対象**: BUZZ BASE 事業戦略策定のための基礎調査

## 1. 競合分析

### 1.1 主要競合サービス一覧

| サービス名 | 運営会社 | 種別 | 対象ユーザー | 野球特化度 |
|-----------|---------|------|------------|-----------|
| TeamHub | Link Sports | チーム管理アプリ | アマチュアスポーツチーム全般 | 低（107競技対応） |
| NPB+ | NPB公認 | プロ野球速報 | プロ野球ファン | 高（観戦向け） |
| スポナビ野球速報 | Yahoo Japan | プロ野球速報 | プロ野球ファン | 高（観戦向け） |
| 野球ノート系アプリ | 各社 | 練習記録 | 選手個人 | 高 |
| BUZZ BASE | 個人開発 | 個人成績ランキング | 中高生野球選手 | 最高 |

### 1.2 TeamHub（最大の間接競合）

- **運営**: Link Sports（2014年設立、東京都千代田区）
- **規模**: 40万チーム以上が導入、約90カ国で利用
- **対応競技**: 107種類（野球、サッカー、バスケなど）
- **主な機能**: イベント管理、出欠確認、スコア共有、統計自動生成、チームコミュニケーション
- **料金プラン**:

| プラン | 月額 | 内容 |
|--------|------|------|
| Free | 無料 | 基本機能のみ |
| Basic | $8.49 | 全統計表示 |
| Plus | $17.49 | 野球・ソフトボール専用機能 |

- **収益モデル**: フリーミアム型サブスクリプション
- **強み**: 大規模ユーザーベース、多競技対応、Apple App Store「Today」タブ掲載実績
- **弱み**: 野球特化ではないため成績管理の深さに限界、ランキング・SNS機能なし

### 1.3 プロ野球速報系アプリ（NPB+、スポナビ野球速報）

- **位置づけ**: App Storeスポーツカテゴリ上位（NPB+ 1位、スポナビ 7位）
- **対象**: プロ野球の観戦ファン向け
- **機能**: 試合速報、選手データ、ニュース配信
- **BUZZ BASEとの競合度**: 低い（対象ユーザーが異なる）
- **参考点**: プロ野球関連は日本のスポーツアプリ市場で最大の需要

### 1.4 野球成績管理・記録系アプリ

アマチュア野球選手向けの成績管理に特化したアプリは、現時点で市場に大きなプレイヤーが存在しない状態。野球スコアブック系のアプリは存在するが、チームの試合記録が主体で「個人成績のランキング共有」という切り口のサービスはBUZZ BASEが独自性を持つ。

### 1.5 競合マッピング

```
                高（野球特化）
                    |
         野球ノート系 | BUZZ BASE ★
                    |
    ────────────────┼────────────────
   チーム管理        |        個人成績・SNS
                    |
         TeamHub    | （空白地帯）
                    |
                低（汎用スポーツ）
```

**BUZZ BASEのポジション**: 「野球特化 x 個人成績・SNS」の象限は競合が極めて少ない空白地帯。

| 2年生 | 41,227人 |

| 項目 | 推定値 |

| 中学硬式野球（ボーイズ・シニアなど） | 約3万〜4万人 |

※ 中体連の加盟部員数は2009年の約30万人から大幅減少。少子化と部活動の地域移行の影響が大きい。

### 2.2 日本のスポーツテック市場

- 日本のスポーツテック市場は成長期にあり、DX推進の流れでアマチュアスポーツへのIT導入が加速
- 政府の「スポーツDX」推進政策が追い風

- スポーツテック全体の国内市場は数百億円規模と推定（グローバルでは数兆円規模）

| Nike Run Club（参考） | 無料+ブランド広告 | 無料 | ランナー |

### 3.2 中高生向けアプリの広告・課金特性

**広告モデルの課題と機会**:
- 中高生は購買力が限定的 → 広告単価（CPM/CPC）は一般成人向けより低い傾向
- ただし、スポーツ用品メーカー（ミズノ、SSK、ゼット、ローリングスなど）は中高生野球選手への訴求ニーズが高い
- 推定広告単価: バナー広告 CPM 100〜300円、動画リワード広告 CPM 500〜1,500円
- MAU 100の場合の広告収入は月額数百円程度 → スケールが必要

**サブスクリプションモデルの可能性**:
- 保護者課金モデル: 子供の成績管理に価値を感じる保護者がターゲット
- チーム課金モデル: 監督・コーチ向けの分析機能
- 想定価格帯: 月額300〜500円（個人）/ 月額1,000〜3,000円（チーム）

## 4. BUZZ BASEの差別化ポイント（SWOT分析）

### Strengths（強み）

1. **唯一無二のポジショニング**: 「個人成績 x ランキング x SNS」の組み合わせは競合不在
2. **野球完全特化**: 打率・防御率・OPSなど野球固有の指標を自動計算
3. **無料で利用可能**: 参入障壁が低く中高生が気軽に使える
4. **個人開発の機動性**: 大企業にない速度でユーザーフィードバックを反映可能
5. **ランキング機能のゲーミフィケーション効果**: 競争意欲を刺激し継続利用を促進

### Weaknesses（弱み）

1. **MAU 70〜90の小規模**: ネットワーク効果が発揮されていない
2. **個人開発のリソース制約**: マーケティング・開発の両立が困難
3. **Web版のみ**: ネイティブアプリがない（開発中）ことでモバイル体験に限界
4. **知名度の低さ**: SEO・ASO・SNSでの認知が不足
5. **収益基盤なし**: 現時点で収益化未実施

### Opportunities（機会）

1. **ネイティブアプリ化**: プッシュ通知、オフライン対応でエンゲージメント向上
2. **部活動の地域移行**: チーム横断的な成績比較ニーズが増大
3. **高校野球の注目度**: 甲子園を中心とした文化的関心の高さ
4. **スポーツDX政策**: 政府のデジタル化推進が追い風
5. **少子化による「1人あたり投資」増加**: 保護者の教育・スポーツ投資意欲は維持
6. **硬式野球クラブチーム増加**: 中学で硬式を始める選手が増え、成績管理ニーズ上昇

### Threats（脅威）

1. **野球人口の継続的減少**: TAM自体が縮小傾向
2. **TeamHub等の大手参入**: 既存プレイヤーが野球特化機能を追加する可能性
3. **SNSプラットフォームの代替**: Instagram/TikTokで成績共有する文化の定着
4. **中高生のアプリ離れ**: SNS以外のアプリを継続利用するハードルの高さ
5. **個人情報保護規制**: 未成年のデータ取扱いに関する規制強化リスク

## 5. 機会と脅威の詳細分析

### 5.1 市場トレンド

**追い風となるトレンド**:
- **データドリブン野球の浸透**: プロ野球でのデータ活用（トラッキング、セイバーメトリクス）が中高生にも影響
- **部活動の地域移行**: 2023年から段階的に開始。クラブチーム化でチーム間移動が増え、個人単位の成績管理需要が上昇
- **保護者のデジタルリテラシー向上**: 子供の活動をデジタルで把握したいニーズ
- **「見せる・競う」文化**: Z世代のSNSネイティブな行動様式と、ランキング機能の親和性

**逆風となるトレンド**:
- **少子化**: 中学野球部員数はピーク時の半数以下に減少
- **野球離れ**: サッカー、バスケ等との競争、eスポーツの台頭
- **可処分時間の分散**: TikTok、YouTube等との時間の奪い合い

### 5.2 ネイティブアプリ化による機会

| 機能 | Web版の限界 | ネイティブアプリの利点 |
|------|-----------|-------------------|
| プッシュ通知 | 限定的 | リアルタイム通知でDAU向上 |
| オフライン対応 | 不可 | 試合中の即時記録が可能 |
| カメラ連携 | 制限あり | プレー動画の撮影・紐付け |
| ウィジェット | 不可 | ホーム画面で成績確認 |
| App Store掲載 | なし | ASO（App Store最適化）による新規獲得 |

ネイティブアプリ化は、ユーザー体験の向上だけでなく、App Storeという新たな流入チャネルの獲得という点で大きな機会。

## 6. 総括と提言

### BUZZ BASEの市場ポジション

BUZZ BASEは「中高生野球選手向け個人成績ランキングSNS」という、明確な空白地帯に位置する。競合は間接的なものが中心で、直接的な競合はほぼ存在しない。これは大きなチャンスであると同時に、市場が未成熟（ニーズが顕在化していない可能性）であるリスクも含む。

### 最優先アクション

1. **ネイティブアプリのリリース**: App Store掲載による自然流入の獲得が最も効果的な成長ドライバー
2. **MAU 1,000到達**: 現在のMAU 70〜90を最低1,000まで引き上げ、ネットワーク効果の閾値に到達させる
3. **ユーザーの声の収集**: 現在のアクティブユーザーから定性的フィードバックを収集し、PMF（Product-Market Fit）を検証

### 市場機会の評価

中高生野球人口約28〜32万人に対し、SAM約20〜25万人。現在のMAU 70〜90は浸透率0.03%程度であり、成長余地は大きい。ただし、野球人口の減少トレンドを考慮すると、早期にポジションを確立し、ユーザー当たりの価値（LTV）を高める戦略が重要。
