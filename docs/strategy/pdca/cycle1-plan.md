# BUZZ BASE PDCA第1期 戦略計画書

**作成日**: 2026年3月14日
**期間**: 2026年4月〜2026年9月（6ヶ月間）
**テーマ**: ユーザー数拡大 — MAU 80 → 500

---

## エグゼクティブサマリー

BUZZ BASEは「中高生野球選手向け 個人成績ランキングSNS」として、競合が存在しない明確な空白地帯に位置する。TAM 28〜32万人に対し現在のMAU浸透率は0.03%であり、成長余地は極めて大きい。

第1期（6ヶ月）は**ユーザー数拡大**に全リソースを集中する。収益化（広告・サブスク）は仕組みの準備に留め、本格稼働はMAU 500到達後のPhase 2以降とする。

## 4. アクションプラン

> **戦略変更（実データに基づく）**: 当初はSNSマーケティングを最優先としていたが、
> Google Analytics / Search Consoleの分析により、**SEO（計算ツール）が圧倒的な成長エンジン**
> であることが判明。流入の72.5%がオーガニック検索、SNSはほぼゼロ。
> 計算ツールリリース後、新規ユーザーが3-4倍に急増（「ops 計算」で掲載2位）。
> **既に効いている施策の強化を最優先**とし、SNSは並行して立ち上げる方針に変更。

### Phase 1: SEO強化＋広告基盤（4月 — 最重要月）

4月は新入部員シーズン。SEO経由の流入増加とSNS立ち上げを同時に進める。

#### SEO最適化（最優先 — 最もROIが高い）

| # | 施策 | 優先度 | 期待効果 |
|---|------|--------|---------|
| S1 | `/calculation-of-grades` のタイトル・メタディスクリプション改善 | **最優先** | 8,129表示/CTR 1.4%→5%で月+290クリック |
| S2 | `/tools/ops` のタイトル・メタディスクリプション改善 | **最優先** | 4,264表示/CTR 1.4%→5%で月+150クリック |
| S3 | 「打率計算」ページの順位改善（現在8位→3位以内） | **最優先** | 1,022表示の取り込み |
| S4 | 各計算ツールに構造化データ（FAQ/HowTo schema）追加 | 高 | リッチスニペットでCTR向上 |
| S5 | 「防御率 計算」コンテンツ強化（検索ボリューム大だが未獲得） | 高 | 新規インプレッション獲得 |
| S6 | 新規計算ツール追加候補の検討（WAR、得点圏打率等） | 中 | SEO流入チャネルの拡大 |

#### 計算ツール → 会員登録の導線強化（最優先）

| # | 施策 | 優先度 | 狙い |
|---|------|--------|------|
| P1 | 計算ツール利用後のCTA「成績を継続的に記録・管理しませんか？」 | **最優先** | SEO流入→会員登録の転換率向上 |
| P2 | 計算結果の保存機能（要会員登録）の導線設計 | **最優先** | 自然な登録動機の創出 |
| P3 | 計算ツールページに成績カードのサンプル表示 | 高 | サービスの価値を視覚的に訴求 |
| P4 | 招待リンク生成機能の改善 — ワンタップでLINE/SNS共有 | 高 | 登録後のバイラル促進 |

## 7. リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| SNS運用の継続困難（個人開発の負荷） | 高 | 週2投稿にペースダウンしてでも継続。テンプレート化で効率化 |
| 口コミが広がらない | 高 | 成績カード共有・招待機能のUXを徹底改善。1チーム丸ごと導入を狙う |
| ネイティブアプリ開発遅延 | 中 | Web版の最適化で代替。PWA対応も検討 |
| 既存ユーザーの離脱 | 中 | 月1回のユーザーヒアリング。改善要望を迅速に反映 |
| 広告によるUX悪化 | 中 | 広告は控えめに配置。ユーザーの反応を見て調整 |

## 8. PDCAサイクルの運用方法

### Check（効果測定）のタイミング

| 頻度 | 確認項目 |
|------|---------|
| **週次** | SNS投稿実施状況、新規登録数、エンゲージメント率 |
| **月次** | MAU推移、チャネル別新規獲得数、リテンション率、最も効果の高かった施策 |
| **四半期** | KPI達成度の総合評価、次四半期の計画調整 |

### Act（改善）の判断基準

| 状況 | アクション |
|------|-----------|
| 目標の120%以上達成 | 効果の高い施策にリソース集中。次フェーズの前倒し検討 |
| 目標の80-120%達成 | 現在の施策を継続・微調整 |
| 目標の80%未満 | 施策の見直し。チャネル・コンテンツの方向転換を検討 |
| 特定施策の効果が突出 | その施策を深堀り・横展開 |

- **第1回レビュー**: 2026年6月末（3ヶ月経過時点）
- **第2回レビュー**: 2026年9月末（第1期完了時点）
