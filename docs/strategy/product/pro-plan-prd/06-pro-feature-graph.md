# PRD-06: シーズン跨ぎ成績推移グラフ

**作成日**: 2026-05-12
**ステータス**: ドラフト（実装直前に詳細化）
**親ドキュメント**: `../pro-plan-prd-202605.md`

## 概要

複数シーズンを跨いだ自分の成績推移を折れ線グラフで可視化する Pro 機能。
「3年前と比べて確実に成長している」という自己肯定感を生み、JTBD「成長実感」「諦めない」を支える。

---

## 背景・目的

- 戦略ドキュメントのステップ11 で MVP に採用
- ペルソナ全員の共通動機「データで成長を可視化したい」に直撃
- 既存DBのデータのみで実装可能（新規データ不要）
- 既存の `seasons` テーブルを活用

---

## ユーザーストーリー

### US-01: 大輝の打率推移確認

> 大輝は大学3年生の春季リーグ前、自分の打率推移を確認:
> - 高3夏: 0.285
> - 大1春: 0.245（落ち込み）
> - 大1秋: 0.298
> - 大2春: 0.315
> - 大2秋: 0.330
> - 大3春（現在）: 0.342
> 「確実に成長している」と確信し、自信を持って春季リーグに臨む。

### US-02: 翔太の防御率改善履歴

> 翔太は WHIP・防御率・K/BB を同時に折れ線で表示。
> 「春は調子いいが秋は落ちる」傾向を発見、来年の対策を立てる。

## 表示する指標一覧

### 打撃指標

| 指標 | 計算式 | 既存DB |
|----|------|----|
| 打率 | 安打 / 打数 | batting_averages |
| 出塁率 | (安打+四球+死球) / (打数+四球+死球+犠飛) | 同上 |
| 長打率 | 塁打 / 打数 | 同上 |
| OPS | 出塁率 + 長打率 | 同上（計算で算出） |
| ISO | 長打率 - 打率 | 同上 |
| BB/K | 四球 / 三振 | 同上 |
| 本塁打数 | 本塁打 | 同上 |
| 打点 | 打点 | 同上 |

### 投手指標

| 指標 | 計算式 | 既存DB |
|----|------|----|
| 防御率 | 自責点 × 9 / 投球回 | pitching_results |
| WHIP | (被安打+与四球) / 投球回 | 同上 |
| K/9 | 奪三振 × 9 / 投球回 | 同上 |
| BB/9 | 与四球 × 9 / 投球回 | 同上 |
| K/BB | 奪三振 / 与四球 | 同上 |
| FIP | 簡易計算式 | 同上 |
| 勝率 | 勝利 / (勝利+敗戦) | 同上 |

## データモデル

### 既存テーブルを活用（新規テーブル不要）

- `seasons` テーブル → X軸のラベル
- `batting_averages` テーブル → 打撃指標
- `pitching_results` テーブル → 投手指標
- `match_results` テーブル → 紐付け

### 集計クエリ例

```ruby
class SeasonStats
  def self.batting_for_user(user_id, season_id)
    BattingAverage
      .joins(game_result: :match_result)
      .where(user_id: user_id, game_results: { season_id: season_id })
      .group('game_results.season_id')
      .select(
        'SUM(hit) as total_hit',
        'SUM(times_at_bat) as total_at_bat',
        'SUM(base_on_balls) as total_bb',
        'SUM(strike_out) as total_so',
        # ...
      )
  end

  def self.batting_average(stats)
    stats.total_hit.to_f / stats.total_at_bat
  end

  def self.ops(stats)
    obp(stats) + slg(stats)
  end
end
```

## API 設計

### GET /api/v1/stats/season_transition

クエリパラメータ:
- `metrics`: 表示する指標のカンマ区切り（"batting_average,ops,obp"）
- `from_season_id`: 開始シーズン
- `to_season_id`: 終了シーズン
- `type`: "batting" or "pitching"

レスポンス:
```json
{
  "seasons": [
    {
      "season_id": 1,
      "name": "2024年春季",
      "start_date": "2024-04-01",
      "end_date": "2024-06-30",
      "type": "spring"
    },
    ...
  ],
  "metrics": [
    {
      "key": "batting_average",
      "label": "打率",
      "color": "#3B82F6",
      "data": [
        {"season_id": 1, "value": 0.285, "games": 12},
        {"season_id": 2, "value": 0.298, "games": 15},
        ...
      ],
      "best": {"season_id": 5, "value": 0.342}
    },
    ...
  ]
}
```

## エッジケース・エラーハンドリング

| ケース | 対応 |
|------|----|
| シーズンデータが1個のみ | 「2シーズン以上のデータが必要です」表示 |
| 投手データなし（野手のみ） | 投手タブを非表示 |
| データ欠損のシーズン | グラフで点として表示せずスキップ |
| 投球回 0 のシーズン | 防御率・WHIP は計算しない |
| 打数 0 のシーズン | 打率等は計算しない |
